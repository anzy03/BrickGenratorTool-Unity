# BrickGenratorTool - Unity

This is a tool made for Unity to spawn bricks for a brickbreaker style game.

## Dependencies
 - [Easy Buttons.](https://github.com/madsbangh/EasyButtons) 

## Installation.
 - Clone the script and attach to a GameObject.
